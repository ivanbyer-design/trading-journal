⚡ Tradovate / Apex Trader Funding: EDGEFLOW polls your Tradovate account every 5 seconds via REST API. New fills are automatically logged. Get your CID + API key at trader.tradovate.com → Settings → API Access.
Real endpoint: POST https://[live|demo].tradovateapi.com/v1/auth/accesstokenrequest
🔷 Rithmic R|Protocol API: EDGEFLOW connects via Rithmic's WebSocket feed to receive live order fills. Your broker must enable R|API access on your account. Enter your R|Trader Pro credentials below.

Supported Brokers: Apex Trader Funding (Rithmic), Earn2Trade, Topstep (Rithmic), NinjaTrader Brokerage, AMP Futures, Generic Rithmic accounts.
📌 WebSocket endpoint: wss://rituz00100.rithmic.com:443 (paper) or wss://rprotocol.rithmic.com:443 (live)
Protocol uses Protobuf binary messages. EDGEFLOW handles auth → subscribe to order_plant for fill streaming. Your FCM ID is provided by your broker if required.
⚡ TopstepX: TopstepX uses Tradovate as its execution engine. Log in with your TopstepX Tradovate credentials. Available on supported plan tiers.
✍ Manual Mode: No API connection. Log trades manually using the + Log Trade button, or import a CSV export from your platform. All data is stored locally in your browser.
